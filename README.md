SiteShot
========

Single-page web site snapshot script.

Use for SEO in single-page apps (Angular.js, Backbone.js, ...)

## Installation
1. Install Node.js
2. Install SiteShot  
  ```
  npm install siteshot -g
  ```
3. Create config in website dir  
  ```
  siteshot config
  ```
4. Run this util and get snapshots  
  ```
  siteshot
  ```
